# Example Brand System — Dark Editorial

> Customize this file with your own brand values. The creative director agent
> uses this to review your slides.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#0A0A0A` | Near-black, all slide backgrounds |
| Primary text | `#FFFFFF` | Headlines, key content |
| Secondary text | `#9A9A9A` | Subtitles, body copy, labels |
| Accent | `#F2A046` | Hand-drawn marks, emphasis, highlights |
| Functional | `#2E2C2B` | Borders, dividers, grid lines (low opacity) |

**Rules:**
- Never use pure black `#000000` — always near-black for depth
- Accent color is for emphasis only — used sparingly
- No more than 3 colors visible on any single slide
- No gradients, no glow effects, no neon

## Typography

### Headlines
- **Font:** Playfair Display (or similar high-contrast serif)
- **Weight:** Bold/Black for primary words, Regular Italic for concept words
- **Treatment:** Mixed case. Tight tracking (-10 to -20). Period at end of titles.
- **Example:** "Redefining the *Capability* Limits."

### Body Text
- **Font:** Inter, Helvetica Neue, or similar geometric sans
- **Weight:** Regular (400) for body, Medium (500) for emphasis
- **Size relationship:** Headlines ~3x body text size
- **Line length:** 45-60 characters per line
- **Leading:** 1.4x font size

### Labels & Captions
- **Font:** Same sans as body
- **Weight:** Medium
- **Case:** Small caps for section labels ("01 TRIGGER")
- **Color:** Secondary text color

## Layout

- **Grid:** 6-column Swiss grid with 80-100px outer margins
- **Baseline:** 8px or 12px baseline grid for vertical rhythm
- **Alignment:** Left-align headlines and body. Center only for closing/manifesto slides.
- **Negative space:** Generous. Let content breathe. When in doubt, remove an element.

## Accent Treatments

- **Hand-drawn underline:** Amber brush stroke beneath key words. 4-6px weight, slightly irregular.
- **Hand-drawn circle:** Amber brush circle around key data points. Same weight as underline.
- **Corner marks:** Thin gray L-shaped registration marks in each corner. 1-2px stroke.
- **Horizontal rules:** Thin amber rule (1-1.5px) to separate title from content.

**Rules:**
- One accent treatment per slide maximum
- Never combine underline + circle on the same slide
- Hand-drawn elements should feel organic, not perfectly geometric

## Icons & Diagrams

- **Style:** Minimal line art, 2px stroke weight
- **Color:** Muted gray or accent color (not both on same slide)
- **Size:** Large enough to anchor their section (not decorative afterthoughts)
- **No:** 3D effects, drop shadows, gradient fills, stock illustrations

## Slide Types

### Title Slide
- Large serif headline left-aligned
- Subtitle below in secondary color
- Attribution bottom-left
- Corner marks

### Section Divider
- Section label top-left (e.g., "02 THE SHIFT")
- Centered icon (line art)
- Centered headline below icon
- Centered subtitle below headline

### Content Slide
- Title top-left in serif
- Content below, respecting the grid
- One idea per slide

### Data Slide
- Title top-left
- Hero number large and centered
- Supporting context below
- Minimal — let the number speak

### Closing Slide
- Large centered serif headline
- Accent treatment on key word
- Supporting quote/statement below
- Maximum whitespace

## Do's and Don'ts

### Do
- Use consistent margins across every slide
- Let negative space do the work
- Use accent color to guide the eye to one thing per slide
- Make it readable at thumbnail size
- Think "editorial magazine spread"

### Don't
- Add gradients or glow effects
- Use more than 3 colors per slide
- Use stock photography
- Crowd slides with content
- Use different fonts on different slides
- Add decorative elements that don't serve the narrative
